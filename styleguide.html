CSS Architecture

Single-CSS Strategy

ToolsShare uses a single CSS file architecture for optimal performance and maintainability:

File Structure

  • assets/css/main.css - Single stylesheet for entire site
  • CSS Custom Properties (variables) for design tokens
  • BEM methodology with ts- namespace
  • Mobile-first responsive design
  • Low specificity selectors (no !important)

Naming Convention

/* Block */
.ts-card { }

/* Element */
.ts-card__header { }
.ts-card__content { }
.ts-card__footer { }

/* Modifier */
.ts-card--featured { }
.ts-btn--primary { }
.ts-btn--large { }

Extending the System

To add new components:

  1. Use existing design tokens from :root
  2. Follow BEM naming with ts- prefix
  3. Keep specificity low (single class selectors)
  4. Add responsive breakpoints as needed
  5. Test accessibility and keyboard navigation

Consistency Enforcement
